# The-Ascender

The Ascender is a 2D elevator-based game featuring combat and puzzle rooms.
Players progress through multiple floors, each containing unique challenges and mechanics.

## 🎮Features:
- Smooth player movement and interaction system
- Room-based progression (15+ rooms)
- Combat and puzzle mechanics
- Dynamic map/room structure

## ⚒️Technologies:
- Java
- Object Oriented Programming
- Git for team collaboration

## 🚀How to run:
- Clone the repository
- Open the project in IntelliJ IDEA (or any Java IDE)
- Run the main file
